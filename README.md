AI-Based Lung Diagnosis System

An AI-powered system for diagnosing lung diseases using cough sound analysis, chest X-ray images, and text-based inputs. The system combines multiple data sources to provide accurate diagnostic insights.

Features

 Cough sound analysis using AI
 Chest X-ray image classification
 Text input analysis for symptoms
 Multi-input diagnosis system

 Backend Role

Developed using .NET with a 3-tier architecture and clean code principles. Responsible for building RESTful APIs, handling data processing, and integrating AI models.

 🧰 Technologies

 ASP.NET Core
 REST APIs
 SQL Server
 AI Integration
